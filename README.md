# ⛽ Gas Station - 智能知识管家

一个运行在本地的个人知识速记工具，你只需**随意写下灵感或资源**，AI 管家会自动帮你纠错、提炼、分类，并保存为结构化的 Markdown 笔记。所有笔记存储在你的电脑上，大脑则跑在云端（免费使用 DeepSeek API），无需 GPU，轻量又安全。

---

## ✨ 功能亮点

- **📝 自由速记**：像记便签一样输入内容，无需关心格式。
- **🧠 AI 自动整理**：基于 DeepSeek 大模型，自动纠正错别字，提炼核心要点，生成干净漂亮的 Markdown 笔记。
- **📂 智能分类**：AI 会根据内容主题自动创建或选择文件夹（如 `人生经验`、`技术工具`、`读书笔记`）并归档。
- **🔍 全文检索**：随时搜索过往的笔记，快速找到所需信息。
- **💻 完全本地存储**：所有笔记以 `.md` 文件保存在你的电脑上，离线也能看，数据 100% 归你所有。
- **🆓 免费云端大脑**：使用 DeepSeek 提供的免费 API 额度，日常使用完全零成本。

---

## 📸 界面预览

启动后，浏览器会打开一个双栏页面：

- **左侧**：速记输入框、搜索框。
- **右侧**：文件夹分类浏览、笔记阅读、笔记数量统计。

---

## 🛠️ 环境要求

- Python 3.9 或更高版本
- 一个 [DeepSeek 开放平台](https://platform.deepseek.com/) 的 API Key（免费，详见下方获取方式）
- 无需 GPU，普通电脑即可运行
